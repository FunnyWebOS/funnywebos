Boot

Astuce: l'ISO est souvent très lourd. Si ça rame, utilise plutôt un fichier local (ci-dessous) ou une image plus légère.
Le boot se fait sur cdrom. Le fichier n'est jamais uploadé: il reste sur ton PC.
Le mode fast nécessite crossOriginIsolated (COOP/COEP). GitHub Pages ne l'active pas par défaut.
Si tu veux vraiment un "Android dans le navigateur", c'est la voie la plus simple. Par contre: performances limitées + certains ISO Android ne bootent pas toujours.
